# ModProtocols - 模组通信框架

为游戏模组提供标准化通信能力。

## 核心协议：ModHttpV1

- **点对点通信**：模组间双向文本传输
- **异步处理**：基于 UniTask，不阻塞主线程
- **消息队列**：每模组独立队列（最大200条）
- **线程安全**：支持多模组并发通信

## 应用场景

- **跨模组终端操作**：A模组命令调用B模组功能
- **资源数据共享**：模组间查询和交换数据
- **协调功能**：多模组联动实现复杂功能

## 未来扩展

- 二进制传输协议
- 广播/组播协议
- 事件订阅机制

## 特性

- 轻量集成，无侵入设计
- 完善日志，便于调试
- 动态扩展，保持兼容

## 重要说明

目前该功能已经集成进入 Ducky.Sdk，使得每个 mod 都可以是 host。因此除非特殊需要，例如不是使用 Ducky.Sdk 打开的 mod 需要利用通讯协议的，则需要安装。否则不需要安装。
